## 执行摘要

本报告针对某技术团队（未注册公司）向某国有企业提供技术服务过程中可能涉及的合规风险进行系统性评估。

**核心风险结论**：

| 风险等级 | 风险领域 | 核心风险点 |
|:---:|:---|:---|
| 🔴 极高 | 国有企业关联交易合规风险 | 团队成员与国企管理层存在亲属关系，可能触发廉洁从业禁止性规定 |
| 🔴 极高 | 刑事责任风险 | 存在行受贿、洗钱等刑事风险 |
| 🟠 高 | 劳动关系认定风险 | 技术服务关系可能被认定为劳动关系 |
| 🟠 高 | 税务合规风险 | 未注册公司状态下收款存在偷逃税款风险 |
| 🟠 高 | 合同效力风险 | 合同主体资格存在瑕疵 |
| 🟡 中 | 知识产权风险 | 技术成果归属约定不明 |
| 🟡 中 | 数据合规风险 | 数据安全与隐私保护 |
| 🟡 中 | 公司设立风险 | 无照经营风险 |

**首要建议**：立即核实团队成员与国有企业管理人员的具体亲属关系，评估是否触发《国有企业领导人员廉洁从业若干规定》禁止性规定。

## 第二章 核心法律风险分析

### 2.1 合同主体资格风险

#### 2.1.1 风险事项

某技术团队以未注册公司的主体签订技术服务合同，存在合同主体资格瑕疵。

#### 2.1.2 法律依据

| 法规名称 | 相关条款 | 核心内容 |
|:---|:---|:---|
| 《中华人民共和国民法典》 | 第四百六十五条 | 依法成立的合同，受法律保护 |
| 《中华人民共和国民法典》 | 第五百零九条 | 当事人应当按照约定全面履行自己的义务 |
| 《中华人民共和国民法典》 | 第一百四十三条 | 民事法律行为有效要件 |
| 《中华人民共和国公司法》 | 第七条 | 设立公司，应当依法向公司登记机关申请设立登记 |

#### 2.1.3 风险分析

| 风险点 | 具体表现 | 潜在后果 |
|:---|:---|:---|
| 合同效力 | 未注册公司不具有独立的法人资格，不能以公司名义签订合同 | 合同可能被认定为无效或效力待定 |
| 责任承担 | 个人签署的合同，个人承担无限责任 | 团队成员面临个人资产被追索的风险 |
| 主体信任 | 商业交易中面临信任度不足的问题 | 交易相对方可能要求额外担保 |
| 后续争议 | 发生纠纷时，对方可能以主体不适格为由主张合同无效 | 维权难度增加 |

#### 2.1.4 深度分析：合同主体类型对比

| 主体类型 | 合同签订方式 | 责任承担 | 税务处理 | 适用场景 |
|:---|:---|:---|:---|:---|
| 自然人 | 以个人名义签订 | 无限连带责任 | 按劳务报酬所得缴纳个税 | 短期、临时性服务 |
| 个体工商户 | 以字号名称签订 | 无限连带责任 | 按经营所得缴纳个税 | 小规模经营 |
| 有限责任公司 | 以公司名义签订 | 以公司资产为限 | 企业所得税+股东分红个税 | 正规商业运营 |
| 有限合伙企业 | 以企业名义签订 | 普通合伙人无限连带责任 | 企业所得税+合伙人个税 | 风险投资、股权激励 |

#### 2.1.5 防控建议

| 阶段 | 建议措施 | 可行性 | 优先级 |
|:---|:---|:---:|:---:|
| 立即 | 以个人名义签署合同，明确约定权利义务 | ✅ | P1 |
| 短期（1个月内） | 注册成立有限责任公司或有限合伙 | ✅✅ | P1 |
| 中期（3个月内） | 以公司名义重新签订正式合同 | ✅✅ | P1 |
| 长期 | 建立合同管理制度，完善合同审批流程 | ✅✅ | P2 |

### 2.2 国有企业关联交易合规风险

#### 2.2.1 风险事项

团队成员之一为国有企业管理层亲属，可能被认定为关联关系，触发国有企业关联交易审批要求。

#### 2.2.2 法律依据

| 法规名称 | 相关条款 | 核心内容 |
|:---|:---|:---|
| 《中华人民共和国企业国有资产法》 | 第四十三条 | 国家出资企业与其关联方之间不得利用关联交易损害出资人利益 |
| 《中华人民共和国企业国有资产法》 | 第七十二条 | 关联关系，是指公司控股股东、实际控制人、董事、监事或者高级管理人员与其直接或者间接控制的企业之间的关系 |
| 《国有企业领导人员廉洁从业若干规定》 | 2026年修订版 | 禁止国企管理人员配偶、子女从事与本企业相同或关联的业务 |
| 《中华人民共和国公司法》 | 第一百二十四条 | 上市公司董事、监事、高级管理人员与董事会会议决议事项所涉及的企业有关联关系的，不得行使表决权 |

### 2.3 税务合规风险

#### 2.3.1 风险事项

未注册公司状态下收取技术服务费，存在偷逃税款的法律风险。

#### 2.3.2 法律依据

| 法规名称 | 相关条款 | 核心内容 |
|:---|:---|:---|
| 《中华人民共和国增值税法》 | 第十五条 | 在中国境内销售服务、无形资产的单位和个人，为增值税纳税人 |
| 《中华人民共和国企业所得税法》 | 第一条 | 在中国境内，企业和其他取得收入的组织为企业所得税的纳税人 |
| 《中华人民共和国个人所得税法》 | 第二条 | 下列各项个人所得，应当缴纳个人所得税：（二）劳务报酬所得 |
| 《中华人民共和国税收征收管理法》 | 第六十三条 | 纳税人伪造、变造、隐匿、擅自销毁帐簿、记帐凭证，不缴或者少缴应纳税款的，是偷税 |

#### 2.3.3 税务计算分析

**方案一：个人名义收取服务费（劳务报酬所得）**

| 税种 | 计算方式 | 税率/税额 |
|:---|:---|:---|
| 增值税 | 月收入小于10万，免征 | 0 |
| 个人所得税（劳务报酬） | 每月收入≤800元，免征；800<收入≤4000，减除800后按20%计；收入>4000，减除20%后按20%-40%累进 | 20%-40% |
| 预扣预缴 | 由付款方代扣代缴 | — |

**劳务报酬所得税率表**：

| 级数 | 应纳税所得额 | 税率 | 速算扣除数 |
|:---:|:---|:---:|:---:|
| 1 | 不超过20000元 | 20% | 0 |
| 2 | 超过20000元至50000元 | 30% | 2000 |
| 3 | 超过50000元 | 40% | 7000 |

**示例计算（月收入4万元）**：

```
月收入：40,000元
减除费用（20%）：8,000元
应纳税所得额：32,000元
适用税率：30%
应纳税额：32,000 × 30% - 2,000 = 7,600元
实际税负率：7,600 / 40,000 = 19%
```

**方案二：注册公司后以公司名义收取（企业所得税）**

| 税种 | 计算方式 | 税率/税额 |
|:---|:---|:---|
| 增值税 | 小规模纳税人适用3%征收率，月收入10万以下免征 | 0或3% |
| 企业所得税 | 应纳税所得额 × 25% | 25%（小微企业可享受优惠） |
| 股东分红个税 | 税后利润 × 20% | 20% |
| 综合税负 | 约40%-45% | — |

**小规模纳税人优惠政策**：

| 政策内容 | 执行期限 | 适用条件 |
|:---|:---|:---|
| 增值税月销售额10万元以下免征 | 2027年12月31日前 | 小规模纳税人 |
| 企业所得税小微优惠 | 2027年12月31日前 | 应纳税所得额300万元以下 |

**综合税负对比**：

| 方案 | 适用情形 | 月收入4万元 | 年收入48万元 |
|:---|:---|:---:|:---:|
| 个人劳务报酬 | 短期/临时性 | 约7,600元/月 | 约91,200元/年 |
| 公司企业经营 | 长期正规运营 | 视情况而定 | 视情况而定 |

#### 2.3.4 真实案例

**案例一：郭某、刘某逃税案**

| 要素 | 内容 |
|:---|:---|
| **案例名称** | 郭某、刘某逃税案 |
| **案例类型** | 最高人民法院典型案例 |
| **来源** | 最高人民法院 2025年11月24日发布 |

#### 2.4.4 真实案例

**案例一：圣某欢诉江苏某网络科技有限公司确认劳动关系纠纷案**

| 要素 | 内容 |
|:---|:---|
| **案例名称** | 圣某欢诉江苏某网络科技有限公司确认劳动关系纠纷案 |
| **案例类型** | 最高人民法院指导性案例（具有参照效力） |
| **来源** | 最高人民法院公报，2024年12月20日发布 |
| **基本案情** | 外卖骑手圣某欢通过APP注册成为专送骑手。注册过程中被要求讲出"我要成为个体工商户"，并以个体工商户名义与某管理公司签订《项目转包协议》。工作期间需遵守站点考勤管理，按派单接单 |
| **裁判要点** | 平台企业要求劳动者注册为个体工商户后再签订承揽、合作协议，劳动者主张根据实际履行情况认定劳动关系的，人民法院应当在查明事实的基础上，依据相关法律，准确作出认定。对于存在用工事实，构成支配性劳动管理的，应当依法认定存在劳动关系 |
| **裁判结果** | 认定圣某欢与江苏某网络公司存在劳动关系 |
| **效力说明** | 各级法院审理类似案件时应参照 |

**案例二：王某诉北京某文化传媒有限公司劳动争议案**

| 要素 | 内容 |
|:---|:---|
| **案例名称** | 王某诉北京某文化传媒有限公司劳动争议案 |
| **案例类型** | 最高人民法院指导性案例（具有参照效力） |
| **来源** | 最高人民法院公报，2024年12月20日发布 |
| **基本案情** | 王某与北京某文化传媒公司签订《独家经纪合同》，约定合作服务而非劳动合同。王某对收益分配方式具有较强协商权和议价权，无需遵守公司劳动纪律和奖惩办法 |
| **裁判要点** | 经纪公司对从业人员的工作时间、工作内容、工作过程控制程度不强，从业人员无需严格遵守公司劳动管理制度，且对利益分配等事项具有较强议价权的，应当认定双方之间不存在支配性劳动管理，不存在劳动关系 |
| **裁判结果** | 认定双方不存在劳动关系 |
| **实务启示** | 对工作内容、工作时间有较强自主决定权；无需遵守公司劳动纪律和奖惩办法；对利益分配有较强议价权——满足以上条件，可能不构成劳动关系 |

#### 2.4.5 风险分析

| 风险点 | 具体表现 | 潜在后果 |
|:---|:---|:---|
| 劳动管理事实 | 国有企业对团队成员进行日常管理（考勤、工作安排等） | 可能被认定存在劳动关系 |
| 合同形式与实质不符 | 签订技术服务合同，但实际履行中构成劳动关系 | 按实际法律关系认定 |
| 劳动权益 | 一旦认定劳动关系，需缴纳社保、提供劳动保护等 | 增加用工成本 |
| 法律责任 | 国企可能承担用人单位责任 | 面临劳动仲裁和诉讼风险 |

#### 2.4.6 劳动关系与承揽关系对比

| 区分标准 | 劳动关系 | 承揽/合作关系 |
|:---|:---|:---|
| 主体关系 | 从属关系 | 平等关系 |
| 工作时间 | 由用人单位安排 | 自主决定 |
| 工作地点 | 在用人单位场所 | 自己决定 |
| 劳动工具 | 用人单位提供 | 自己提供 |
| 报酬计算 | 按时间（月/日） | 按成果（项目） |
| 纪律约束 | 需遵守规章制度 | 无需遵守 |
| 指挥监督 | 接受用人单位管理 | 独立完成 |
| 社会保险 | 用人单位必须缴纳 | 无需缴纳 |

#### 2.4.7 防控建议

| 阶段 | 建议措施 | 可行性 | 优先级 |
|:---|:---|:---:|:---:|
| 立即 | 明确合同性质为技术服务合同，而非劳动合同 | ✅ | P1 |
| 立即 | 团队成员不接受国企的日常劳动管理 | ✅ | P1 |
| 立即 | 避免接受国企的考勤、排班等管理要求 | ✅ | P1 |
| 短期 | 建立清晰的合作关系约定，明确双方权利义务 | ✅✅ | P2 |
| 短期 | 使用自己的设备、工具完成工作 | ✅✅ | P2 |
| 中期 | 定期收集保存合作过程的证据材料 | ✅✅ | P2 |

### 5.2 案例详细汇编

#### 5.2.1 国有企业关联交易类案例

**赖某民受贿、贪污、重婚案**

- **案例类型**：典型案例（2021年度）
- **来源**：《中国审判》2021年度十大典型案例
- **裁判日期**：2021年1月判决
- **基本案情**：赖某民系某国有企业董事长，利用职务便利为特定关系人谋取利益并收受巨额贿赂
- **裁判要点**：国企领导人员利用职务便利为亲属谋取利益，构成受贿罪，即使形式上通过关联交易进行，仍属违法
- **相关法条**：《刑法》第385条（受贿罪）、第382条（贪污罪）
- **效力说明**：典型案例，具有重要参考价值

**某国企总经理亲属关联交易未披露案**

- **案例类型**：中央纪委国家监委网站公布的典型案例
- **来源**：中央纪委国家监委网站
- **基本案情**：某国有企业总经理明知其子持股的公司向该国企提供技术服务，存在关联关系，但未按规定向组织报告，亦未进行关联交易披露
- **处理结果**：给予党内严重警告处分；撤销总经理职务；责成亲属退出在相关公司持股；追缴违规获取的经济利益
- **风险警示**：国企领导人员有义务主动报告配偶子女的经商情况；关联交易必须依法进行信息披露和合规审查

#### 5.2.2 税务违法类案例

**郭某、刘某逃税案**

- **案例类型**：最高人民法院典型案例
- **来源**：最高人民法院 2025年11月24日发布
- **基本案情**：郭某、刘某注册成立索某金属制品公司，在无真实交易情况下，从多家公司取得增值税专用发票共计880份用于抵扣税款，税额2300余万元
- **裁判结果**：天津市第一中级人民法院认定二人以欺骗手段进行虚假纳税申报，逃避缴纳税款数额巨大且占应纳税额百分之三十以上，构成逃税罪。判处郭某有期徒刑六年，刘某有期徒刑二年
- **裁判要点**：逃税罪与虚开增值税专用发票罪的界分，关键在于行为人主观上是基于骗取国家税款的故意，还是基于逃避纳税义务的故意
- **相关法条**：《刑法》第201条（逃税罪）

**邓某偷逃税案**

- **案例类型**：税务违法案例
- **来源**：国家税务总局上海税务局 2022年
- **查处日期**：2022年3月
- **基本案情**：演员邓某在2019年至2020年期间，通过虚构业务转换收入性质进行虚假申报，偷逃个人所得税4765.82万元，其他少缴个人所得税1399.32万元
- **处理结果**：追缴税款，加收滞纳金并处罚款共计1.06亿元
- **典型意义**：将劳务报酬所得转换为个体工商户经营所得是常见的偷税手段，但属违法行为

#### 5.2.3 劳动关系认定类案例

**圣某欢诉江苏某网络科技有限公司确认劳动关系纠纷案**

- **案例类型**：最高人民法院指导性案例（具有参照效力）
- **来源**：最高人民法院公报，2024年12月20日发布
- **发布/判决日期**：2024年12月20日发布；一审判决2021年8月2日
- **基本案情**：外卖骑手圣某欢通过APP注册成为专送骑手。注册过程中被要求讲出"我要成为个体工商户"，并以个体工商户名义与某管理公司签订《项目转包协议》。工作期间需遵守站点考勤管理，按派单接单
- **裁判要点**：平台企业要求劳动者注册为个体工商户后再签订承揽、合作协议，劳动者主张根据实际履行情况认定劳动关系的，人民法院应当在查明事实的基础上，依据相关法律，准确作出认定。对于存在用工事实，构成支配性劳动管理的，应当依法认定存在劳动关系
- **裁判结果**：认定圣某欢与江苏某网络公司存在劳动关系
- **效力说明**：各级法院审理类似案件时应参照

**王某诉北京某文化传媒有限公司劳动争议案**

- **案例类型**：最高人民法院指导性案例（具有参照效力）
- **来源**：最高人民法院公报，2024年12月20日发布
- **发布/判决日期**：2024年12月20日发布；一审判决2022年11月25日，二审判决2023年9月5日
- **基本案情**：王某与北京某文化传媒公司签订《独家经纪合同》，约定合作服务而非劳动合同。王某对收益分配方式具有较强协商权和议价权，无需遵守公司劳动纪律和奖惩办法
- **裁判要点**：经纪公司对从业人员的工作时间、工作内容、工作过程控制程度不强，从业人员无需严格遵守公司劳动管理制度，且对利益分配等事项具有较强议价权的，应当认定双方之间不存在支配性劳动管理，不存在劳动关系
- **裁判结果**：认定双方不存在劳动关系
- **效力说明**：各级法院审理类似案件时应参照

## 第六章 结论与建议

### 6.1 总体评估

本次技术服务交易存在**较高的法律风险**，主要风险点包括：

1. **合同主体资格瑕疵**：以未注册公司签订合同，存在效力风险
2. **关联交易风险**：团队成员与国企管理层存在亲属关系，可能触发关联交易认定
3. **刑事责任风险**：存在行受贿的潜在风险
4. **劳动关系认定风险**：可能被认定为事实劳动关系
5. **税务合规风险**：收款主体和纳税方式需要规范

### 6.2 核心建议

| 优先级 | 建议 | 说明 |
|:---:|:---|:---|
| P1 | **尽快注册公司** | 这是解决所有问题的根本，建议在1个月内完成 |
| P1 | **亲属关系主动披露** | 团队成员应主动向国企披露亲属关系，避免潜在利益冲突 |
| P1 | **核实具体亲属关系** | 评估是否触发《国有企业领导人员廉洁从业若干规定》禁止性规定 |
| P2 | **规范合同文本** | 签订明确的技术服务合同，避免被认定为劳动关系 |
| P2 | **税务合规先行** | 咨询税务顾问，规范纳税方式 |
